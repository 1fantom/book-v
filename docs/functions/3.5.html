3.5 Разложение многочлена на множители через производную

Идея через производную

Разложение многочлена на множители тесно связано с поиском его корней. Если многочлен \(P(x)\) имеет корень \(x = a\), то по теореме Безу многочлен делится на \((x - a)\):

\(P(x) = (x - a) Q(x),\)

где \(Q(x)\) — многочлен степени на 1 меньше, чем \(P(x)\).

Чтобы найти корень \(a\), иногда полезно изучить поведение функции \(P(x)\) через её производную \(P'(x)\). Производная показывает скорость изменения многочлена, а точки, где \(P(x) = 0\), — это потенциальные множители.

Общий многочлен степени \(n\)

Рассмотрим многочлен степени \(n\):

\(P_n(x) = a_n x^n + a_{n-1} x^{n-1} + \dots + a_1 x + a_0, \quad a_n \neq 0.\)

Если \(P_n(x)\) имеет корни \(x = r_1, r_2, \dots, r_k\) (учитывая кратность), то он раскладывается в виде произведения линейных множителей:

\(P_n(x) = a_n (x - r_1)^{m_1} (x - r_2)^{m_2} \cdots (x - r_k)^{m_k},\quad \sum_{i=1}^k m_i = n.\)

Зная производную \(P_n'(x)\), мы можем определить интервалы возрастания и убывания, а также найти экстремумы, что помогает локализовать корни многочлена.

Пример

Рассмотрим многочлен

\(P(x) = x^3 - 6x^2 + 11x - 6.\)

Его производная равна

\(P'(x) = 3x^2 - 12x + 11.\)

Для разложения найдем корни многочлена \(P(x)\), проверяя значения, где \(P(x) = 0\):

  • \(P(1) = 1 - 6 + 11 - 6 = 0\), значит, \(x = 1\) — корень.
  • По теореме Безу \(P(x)\) делится на \((x - 1)\).

Делим \(P(x)\) на \((x - 1)\) и получаем

\(P(x) = (x - 1)(x^2 - 5x + 6).\)

Дальше разлагаем квадратный трёхчлен:

\(x^2 - 5x + 6 = (x - 2)(x - 3).\)

Итого разложение:

\(P(x) = (x - 1)(x - 2)(x - 3).\)

Роль производной

Производная помогает понять форму графика многочлена, найти экстремумы и точки перегиба. Зная корни производной \(P'(x) = 0\), мы можем приблизительно определить интервалы, где \(P(x)\) меняет знак, и найти корни \(P(x)\). Это упрощает процесс разложения.
