3.1 Неопределенный интеграл и его свойства

Понятие неопределённого интеграла

Неопределённый интеграл — это одно из фундаментальных понятий математического анализа, тесно связанное с понятием первообразной функции. Для заданной функции \(f(x)\) неопределённым интегралом называется множество всех её первообразных, то есть всех таких функций \(F(x)\), что производная \(F'(x)\) совпадает с исходной функцией \(f(x)\).

Записывается неопределённый интеграл так:

\(\displaystyle \int f(x)\,dx = F(x) + C\),

где \(F(x)\) — первообразная функции \(f(x)\), а \(C\) — произвольная постоянная интегрирования, учитывающая все сдвиги графика \(F(x)\) по вертикали.

Геометрический смысл

Геометрически неопределённый интеграл можно представить как семейство кривых, наклон касательных к которым в каждой точке равен значению функции \(f(x)\) в этой точке. Иными словами, если график функции \(f(x)\) задаёт скорость изменения величины, то графики всех функций \(F(x) + C\) показывают величины, меняющиеся с этой скоростью.

Основные свойства неопределённого интеграла

Неопределённый интеграл обладает важными свойствами, которые существенно облегчают вычисления и анализ функций.

  • Линейность: Интеграл от суммы функций равен сумме интегралов, а константу можно вынести за знак интеграла:
    \(\displaystyle \int [a f(x) + b g(x)]\,dx = a \int f(x)\,dx + b \int g(x)\,dx\), где \(a,b\) — константы.
  • Интеграл константы: Интеграл от постоянной функции равен произведению этой константы на переменную интегрирования:
    \(\displaystyle \int c\,dx = cx + C\), где \(c\) — постоянное число.
  • Сдвиг аргумента: При сдвиге функции вдоль оси \(x\), неопределённый интеграл изменяется аналогично:
    \(\displaystyle \int f(x+a)\,dx = F(x+a) + C\).
  • Производная интеграла: Производная первообразной возвращает исходную функцию:
    \(\displaystyle \frac{d}{dx}\int f(x)\,dx = f(x)\).

Связь с определённым интегралом

Неопределённый интеграл тесно связан с определённым интегралом, который вычисляет площадь под графиком функции на заданном интервале. Если \(F(x)\) — первообразная \(f(x)\), то по формуле Ньютона-Лейбница:

\(\displaystyle \int_a^b f(x)\,dx = F(b) - F(a)\).

Это связывает вычисление площадей с поиском первообразных и подчёркивает важность изучения неопределённых интегралов.

Примеры первообразных

  • Если \(f(x) = x^n\), где \(n \neq -1\), то
    \(\displaystyle \int x^n\,dx = \frac{x^{n+1}}{n+1} + C\).
  • Для функции \(f(x) = \frac{1}{x}\) (при \(x \neq 0\)):
    \(\displaystyle \int \frac{1}{x} \, dx = \ln|x| + C\).
  • Для тригонометрических функций, например, \(f(x) = \cos x\):
    \(\displaystyle \int \cos x\, dx = \sin x + C\).
