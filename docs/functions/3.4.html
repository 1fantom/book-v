3.4 Интегрирование простейших дробей

Определение простейших дробей

Простейшие дроби — это дроби вида \(\displaystyle \frac{A}{x - a}\) или \(\displaystyle \frac{Bx + C}{(x - a)^n}\), где \(A, B, C, a\) — константы, а \(n\) — натуральное число.

Интегрирование таких дробей часто сводится к применению элементарных формул и замене переменных.

Основные формулы

  • Интеграл от простой дроби с линейным знаменателем:
    \(\displaystyle \int \frac{dx}{x - a} = \ln|x - a| + C.\)
  • Интеграл от дроби с квадратным знаменателем:
    \(\displaystyle \int \frac{dx}{(x - a)^n} = \frac{-(x - a)^{1-n}}{n-1} + C, \quad n \neq 1.\)

Пример 1

Вычислим интеграл:

\(\displaystyle \int \frac{dx}{x - 2}.\)

Используем формулу:

\(\displaystyle \int \frac{dx}{x - 2} = \ln|x - 2| + C.\)

Пример 2

Вычислим интеграл:

\(\displaystyle \int \frac{dx}{(x + 1)^3}.\)

По формуле для \(n=3\):

\(\displaystyle \int \frac{dx}{(x + 1)^3} = \frac{-(x + 1)^{-2}}{2} + C = -\frac{1}{2(x+1)^2} + C.\)
