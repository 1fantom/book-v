1.1 Определение функции. Способы задания функций

Функция — это математическое отношение, которое каждому элементу множества X ставит в соответствие ровно один элемент множества Y. Множество X называется областью определения функции, а множество значений функции — областью значений.

Обозначается функция часто как \( f: X \to Y \), где для каждого \( x \in X \) определено значение \( f(x) \in Y \).

Функция играет важнейшую роль во всех разделах математики и её приложениях, поскольку она описывает зависимость между переменными и позволяет моделировать реальные процессы.

Основные способы задания функции

  • Явная форма: функция задана формулой \( y = f(x) \). Например, \( y = 2x + 3 \) — линейная функция.
  • Неявная форма: функция задана уравнением, связывающим \( x \) и \( y \), например \( F(x,y) = 0 \). Пример: окружность \( x^2 + y^2 - 1 = 0 \).
  • Табличный способ: значения функции заданы в виде таблицы соответствий для конечного или счётного множества аргументов.
  • Графический способ: функция представлена графиком — множеством точек на плоскости.
  • Описательный способ: словесное описание правила соответствия.

Пример функции

Рассмотрим функцию \( f(x) = \sqrt{x} \), определённую на множестве всех неотрицательных чисел \( x \geq 0 \). Для каждого такого \( x \) находится число \( y \), равное квадратному корню из \( x \).

\( f: [0, +\infty) \to [0, +\infty), \quad f(x) = \sqrt{x} \)

Свойства функции

Для изучения функции важно знать:

  • Область определения — все значения \( x \), при которых функция имеет смысл.
  • Область значений — множество значений \( y \), которые принимает функция.
  • Поведение функции — как она меняется при изменении аргумента.

Понимание этих свойств позволяет анализировать функции и использовать их для решения прикладных задач.
