3.7 Простейшие тригонометрические интегралы

Основные формулы

Тригонометрические функции часто встречаются в интегралах, и их первообразные известны и широко используются. Вот базовые формулы интегрирования простейших тригонометрических функций:

  • Интеграл синуса:
    \(\displaystyle \int \sin x\, dx = -\cos x + C\)

    Объяснение: Производная косинуса равна минус синус, поэтому при интегрировании \(\sin x\) мы получаем \(-\cos x\).

  • Интеграл косинуса:
    \(\displaystyle \int \cos x\, dx = \sin x + C\)

    Объяснение: Производная синуса равна косинусу, поэтому интеграл от \(\cos x\) равен \(\sin x\).

  • Интеграл тангенса:
    \(\displaystyle \int \tan x\, dx = -\ln|\cos x| + C\)

    Объяснение: Тангенс — это \(\frac{\sin x}{\cos x}\), а производная \(\cos x\) — \(-\sin x\), поэтому при замене переменной получаем логарифмическую форму.

  • Интеграл котангенса:
    \(\displaystyle \int \cot x\, dx = \ln|\sin x| + C\)

    Объяснение: Котангенс — это \(\frac{\cos x}{\sin x}\), и аналогично интеграл получается через логарифм синуса.

  • Интеграл секанса:
    \(\displaystyle \int \sec x\, dx = \ln|\sec x + \tan x| + C\)

    Объяснение: Формула получается, если умножить подынтегральное выражение на \(\frac{\sec x + \tan x}{\sec x + \tan x}\) и применить подстановку.

  • Интеграл косеканса:
    \(\displaystyle \int \csc x\, dx = -\ln|\csc x + \cot x| + C\)

    Объяснение: Аналогично интегралу секанса, используется приём умножения на выражение с косекансом и котангенсом для удобной замены.

Примеры

Применим эти формулы на простых примерах, чтобы закрепить понимание:

  • \(\displaystyle \int \sin 2x\, dx\)

    Сделаем замену \(t = 2x\), тогда \(dt = 2 dx\), следовательно, \(dx = \frac{dt}{2}\). Интеграл становится:

    \(\int \sin 2x\, dx = \frac{1}{2} \int \sin t\, dt = -\frac{1}{2} \cos t + C = -\frac{1}{2} \cos 2x + C\)
  • \(\displaystyle \int \cos 3x\, dx\)

    Заменяем \(t = 3x\), \(dt = 3 dx\), \(dx = \frac{dt}{3}\). Получаем:

    \(\int \cos 3x\, dx = \frac{1}{3} \int \cos t\, dt = \frac{1}{3} \sin t + C = \frac{1}{3} \sin 3x + C\)
  • \(\displaystyle \int \tan x\, dx\)

    Используем формулу напрямую:

    \(\int \tan x\, dx = -\ln|\cos x| + C\)

Важно помнить

При интегрировании функций с аргументом, умноженным на константу (\(ax\)), всегда учитывайте правило замены переменной — коэффициент \(a\) нужно делить вне интеграла. Это поможет избежать ошибок и сделает вычисления правильными.
