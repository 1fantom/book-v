3.3 Интегрирование по частям

Основная формула интегрирования по частям

Интегрирование по частям — это метод вычисления интегралов, основанный на формуле производной произведения функций. Если функции \(u = u(x)\) и \(v = v(x)\) имеют непрерывные производные, то:

\(\displaystyle \int u(x) \, dv = u(x)v(x) - \int v(x) \, du.\)

Здесь \(du = u'(x) dx\), \(dv = v'(x) dx\).

Объяснение метода

Идея метода состоит в том, чтобы представить подынтегральное выражение в виде произведения двух функций, где одну из них удобно дифференцировать, а другую — интегрировать. После этого интеграл преобразуется в более простой или удобный для вычисления.

Алгоритм применения

  • Выбираем функции \(u(x)\) и \(dv\) так, чтобы дифференцирование \(u\) упростило выражение, а интегрирование \(dv\) было простым.
  • Вычисляем дифференциал \(du = u'(x) dx\) и интеграл \(v = \int dv\).
  • Подставляем в формулу интегрирования по частям.
  • Вычисляем полученный интеграл, надеясь, что он проще исходного.

Пример

Рассмотрим интеграл:

\(\displaystyle \int x \cos x\, dx.\)

Выберем:

  • \(u = x\), тогда \(du = dx\),
  • \(dv = \cos x\, dx\), тогда \(v = \sin x\).

Подставляем в формулу интегрирования по частям:

\(\displaystyle \int x \cos x\, dx = x \sin x - \int \sin x\, dx.\)

Вычисляем оставшийся интеграл:

\(\displaystyle \int \sin x\, dx = -\cos x + C.\)

Итог:

\(\displaystyle \int x \cos x\, dx = x \sin x + \cos x + C.\)

Советы при выборе функций

  • Часто \(u\) выбирают как функцию, которую легко дифференцировать (например, полиномы, логарифмы, арктангенсы).
  • \(dv\) — функция, интеграл которой известен или легко находится (тригонометрические, экспоненциальные функции).
  • Если результат не упрощается, попробуйте другой выбор \(u\) и \(dv\).
