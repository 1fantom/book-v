3.8 Интегрирование простейших иррациональностей

Понятие иррациональных функций

Иррациональные функции — это функции, содержащие корни, например, \(\sqrt{x}\), \(\sqrt{ax + b}\), или более сложные выражения с корнями. Интегрирование таких функций требует применения специальных приёмов и формул.

Основные формулы интегрирования

  • Интеграл от корня степени \(n\):
    \(\displaystyle \int x^{\frac{m}{n}}\, dx = \frac{n}{m + n} x^{\frac{m+n}{n}} + C, \quad m \neq -n\)

    Здесь \(\frac{m}{n}\) — дробный показатель степени, например \(\frac{1}{2}\) для квадратного корня.

  • Интеграл от корня вида \(\sqrt{ax + b}\):
    \(\displaystyle \int \sqrt{ax + b}\, dx = \frac{2}{3a}(ax + b)^{\frac{3}{2}} + C\)

    Этот результат получается благодаря замене переменной \(t = ax + b\).

  • Интеграл от обратного корня:
    \(\displaystyle \int \frac{dx}{\sqrt{ax + b}} = \frac{2}{a} \sqrt{ax + b} + C\)

Примеры интегрирования

  • \(\displaystyle \int \sqrt{x}\, dx = \int x^{\frac{1}{2}} dx = \frac{2}{3} x^{\frac{3}{2}} + C\)
  • \(\displaystyle \int \sqrt{3x + 4}\, dx = \frac{2}{9} (3x + 4)^{\frac{3}{2}} + C\)

    Здесь использована формула с \(a=3, b=4\).

  • \(\displaystyle \int \frac{dx}{\sqrt{5x - 2}} = \frac{2}{5} \sqrt{5x - 2} + C\)

Метод подстановки

В большинстве случаев для интегрирования иррациональностей удобно использовать замену переменной \(t = ax + b\). После замены интеграл принимает вид, с которым легче работать:

\(\displaystyle \int R(\sqrt{ax + b})\, dx \to \int R(\sqrt{t}) \frac{dt}{a}\)

где \(R\) — функция от корня. Такой приём упрощает вычисления и помогает избежать ошибок.
