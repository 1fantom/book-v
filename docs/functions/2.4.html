2.4 Производные основных элементарных функций

Теоретические основы

Производная функции в данной точке характеризует скорость изменения функции в этой точке и определяется как предел отношения приращения функции к приращению аргумента при стремлении последнего к нулю:

\[ f'(x) = \lim_{h \to 0} \frac{f(x + h) - f(x)}{h} \]

Элементарные функции — это простейшие функции, часто встречающиеся в математике и естественных науках. К ним относятся степенные функции, показательные, логарифмические, тригонометрические и их обратные. Производные этих функций играют ключевую роль при решении множества задач математического анализа.

Ниже приведены основные правила дифференцирования для наиболее часто используемых функций. Эти формулы следует знать наизусть, поскольку они являются фундаментом для более сложных вычислений.

Производные элементарных функций

\[ \begin{aligned} &\text{Постоянная:} && \frac{d}{dx}(c) = 0 \\ &\text{Степенная функция:} && \frac{d}{dx}(x^n) = nx^{n-1} \\ &\text{Синус:} && \frac{d}{dx}(\sin x) = \cos x \\ &\text{Косинус:} && \frac{d}{dx}(\cos x) = -\sin x \\ &\text{Тангенс:} && \frac{d}{dx}(\tan x) = \frac{1}{\cos^2 x}, \quad x \ne \frac{\pi}{2} + \pi n \\ &\text{Показательная функция:} && \frac{d}{dx}(e^x) = e^x \\ &\text{Натуральный логарифм:} && \frac{d}{dx}(\ln x) = \frac{1}{x}, \quad x > 0 \end{aligned} \]

Эти формулы применяются как для вычислений вручную, так и при использовании более сложных правил, таких как правило произведения, частного и сложной функции (правило цепочки).

Графическая интерпретация

Геометрически производная функции в точке — это угловой коэффициент касательной к графику функции в этой точке. Если производная положительна, функция возрастает; если отрицательна — убывает. Если производная равна нулю — это может быть максимум, минимум или точка перегиба.

\[ f'(x) > 0 \Rightarrow \text{функция возрастает}, \quad f'(x) < 0 \Rightarrow \text{функция убывает} \]

Примеры применения

Пример 1: Найдём производную функции \( f(x) = 5x^3 \).
Ответ: \( f'(x) = 15x^2 \)
Пример 2: \( f(x) = \ln(x^2 + 1) \). Используем правило сложной функции:
Ответ: \( f'(x) = \frac{2x}{x^2 + 1} \)
Пример 3: \( f(x) = x \cdot e^x \). Используем правило произведения:
Ответ: \( f'(x) = x \cdot e^x + e^x = e^x(x + 1) \)

Пример использования

Пример: Найдём производную функции \( f(x) = 3x^2 + \sin x \).
Ответ: \( f'(x) = 6x + \cos x \)
