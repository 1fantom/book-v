4.6 Формула Ньютона-Лейбница

Суть формулы

Формула Ньютона-Лейбница связывает определённый интеграл функции с её первообразной. Если \( F(x) \) — первообразная функции \( f(x) \), то определённый интеграл на отрезке \([a, b]\) выражается как разность значений первообразной в концах отрезка:

\[ \int_a^b f(x) \, dx = F(b) - F(a) \]

Здесь \( F'(x) = f(x) \), то есть производная первообразной равна исходной функции.

Доказательство (интуитивное)

Рассмотрим функцию \( G(x) = \int_a^x f(t) \, dt \). По теореме о производной интеграла с переменным верхним пределом, \( G'(x) = f(x) \). Значит, \( G(x) \) — первообразная функции \( f(x) \). Следовательно, существует константа \( C \) такая, что:

\[ G(x) = F(x) + C \]

Подставляя \( x = a \), получаем:

\[ G(a) = \int_a^a f(t) \, dt = 0 = F(a) + C \implies C = -F(a) \]

Значит,

\[ G(x) = F(x) - F(a) \]

При \( x = b \) имеем:

\[ \int_a^b f(t) \, dt = G(b) = F(b) - F(a) \]

Пример

Найдём определённый интеграл:

\[ \int_0^1 (3x^2 + 2x) \, dx \]

Первообразная:

\[ F(x) = x^3 + x^2 + C \]

Применяем формулу Ньютона-Лейбница:

\[ \int_0^1 (3x^2 + 2x) \, dx = F(1) - F(0) = (1 + 1) - (0 + 0) = 2 \]

Значение формулы

Формула Ньютона-Лейбница — ключевой инструмент вычисления определённых интегралов, позволяющий заменить сложную операцию интегрирования на простое вычисление значений первообразной на концах интервала.

Это одна из причин, почему изучение первообразных — важнейшая часть курса математического анализа.
