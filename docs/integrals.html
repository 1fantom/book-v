Интегралы

1. Неопределённый интеграл

Неопределённый интеграл функции \(f(x)\) — это семейство всех её первообразных \(F(x)\):

$$\int f(x)\,dx = F(x) + C,\quad F'(x)=f(x).$$

Константа \(C\) называется константой интегрирования.

2. Определённый интеграл

Определённый интеграл от \(a\) до \(b\) вычисляется по формуле Ньютона–Лейбница:

$$\int_{a}^{b} f(x)\,dx = F(b) - F(a),$$

где \(F\) — любая первообразная \(f\) на \([a,b]\).

3. Основная теорема анализа

Связывает дифференцирование и интегрирование: если \(F\) — первообразная \(f\), то интегрирование и затем дифференцирование возвращает исходную функцию:

$$\frac{d}{dx}\Bigl(\int_{a}^{x}f(t)\,dt\Bigr)=f(x).$$

4. Методы интегрирования

  • Замена переменной: \(u=g(x)\), тогда \(\int f(g(x))g'(x)\,dx = \int f(u)\,du\).
  • Интегрирование по частям:
    • $$\int u\,dv = u v - \int v\,du.$$
  • Разложение на простейшие дроби: применяется для рациональных функций.
  • Тригонометрические подстановки: при корнях вида \(\sqrt{a^2 - x^2}\), \(\sqrt{a^2 + x^2}\) и т.д.

5. Производные и интегралы элементарных функций

  • \(\int x^n\,dx = \tfrac{x^{n+1}}{n+1} + C\) для \(n\neq -1\).
  • \(\int e^x\,dx = e^x + C.\)
  • \(\int \frac{dx}{x} = \ln|x| + C.\)
  • \(\int \sin x\,dx = -\cos x + C,\quad \int \cos x\,dx = \sin x + C.\)

6. Примеры вычислений

Пример 1. \(\int (3x^2 - 5x + 2)\,dx = x^3 - \tfrac{5}{2}x^2 + 2x + C.\)

Пример 2. \(\int x e^x\,dx\) по частям (возьмём \(u=x, dv=e^x dx\)):

\(\int x e^x\,dx = x e^x - \int e^x\,dx = e^x(x - 1) + C.\)

Пример 3. \(\int \frac{dx}{x^2+1} = \arctan x + C.\)

7. Приложения интегралов

  • Вычисление площадей под кривыми \(y=f(x)\).
  • Нахождение объёма тел вращения (формула Гюйгенса–Паппе):
    • $$V = \pi\int_{a}^{b}[f(x)]^2\,dx.$$
  • Применения в физике: работа силы \(W=\int F\,dx\), центр масс и др.
